# ArtWork
Art Space is a simple Android app built with Kotlin and Jetpack Compose. The app displays a curated gallery of artworks, including the title, artist, year, and a detailed description. Users can explore each piece and learn about the artist behind it.

View different artworks with image, title, artist, and year.
Read historical background and biography for each artist.
Switch between artworks using navigation buttons.
Clean and modern UI using Material 3.
